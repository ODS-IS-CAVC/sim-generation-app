# フロントエンドアプリ

## 1.環境前提

| 主なライブラリ | バージョン |
|----------------|------------|
| Vue            | 3.3.4      |
| Vuetify        | 3.6.7      |
| Vite           | 4.2.3      |

## 2.設定のカスタマイズ
[**.env**](./config/env/) の設定ファイルを環境に合わせて修正してください。

## 3.利用方法

### セットアップ
```sh
npm install
```

### コンパイルおよび起動
```sh
npm run {loc | dev | stg | prd}
```
注：環境変数のlocはローカルを意味する。
### コンパイルおよびデプロイ用圧縮
```sh
npm run build:{loc | dev | stg | prd}
```
